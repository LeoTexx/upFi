# 🔧 Run Locally

Clone the project

```bash
  git clone git@github.com:git@github.com:LeoTexx/upFi.git
```

Go to the project directory

```bash
  cd upFi
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
